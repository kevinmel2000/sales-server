# sales-server
Sales API

import the mysql db file at DB folder local mysql

set the credentials at .env file

## Initial setup

### 1. Install the node modules.

```bash
# Install dependencies
yarn
```

### 2. Run the server
```bash
yarn start
```

### 3. API URL
```bash
http://localhost:4000
```
or the deployed version at Heroku
```bash
http://localhost:4000
```
```bash
https://sales-backend.herokuapp.com
```
